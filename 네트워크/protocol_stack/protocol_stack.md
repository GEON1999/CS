# 네트워크를 이루는 장치
- 네트워크는 여러 개의 네트워크 기기를 기반으로 구축됨
- 네트워크 기기는 계층별로 처리 범위를 나눌 수 있음
- 상위 계층을 처리하는 기기는 하위 계층을 처리할 수 있음
- 반대로 하위 계층은 상위 계층을 처리할 수 없음

</br>
